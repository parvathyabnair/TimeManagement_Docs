# Kebab-menu 

Het Kebab-menu (⋮) bevindt zich in de **rechterbovenhoek van de Kopsectie** en biedt snelle toegang tot belangrijke navigatie-items.

### Doel:
* Biedt een alternatieve navigatiemethode voor de zijbalk.
* Verbetert de bruikbaarheid op kleinere schermen en compacte lay-outs.
* Maakt snelle toegang mogelijk zonder het volledige menu te openen.

### Menu-items:
De volgende opties zijn beschikbaar in het Kebab-menu:
* Dashboard
* Urenstaat
* Activiteiten
* Mijn Taken
* Alle Taken
* Projecten
* Projectupdates
* Over Ons
* Instellingen
